#Intro To Razor
###Setup

 Clone repository and `cd` into it
 ```bash
 dotnet restore
 cd .\src\IntroToRazor.DAL
 dotnet ef update database
 cd ..\IntroToRazor
 dotnet run
 ```

2. Home Views, Controllers, and styling
3. Vendor Views, Controllers, and Models
4. Product Controllers and Models
